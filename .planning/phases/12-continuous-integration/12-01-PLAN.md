# Phase 12 Plan 01: GitHub Actions CI

## Objective

Add a GitHub Actions workflow that runs `npm test` (bash syntax + function defs + CLI tests) on every push to main and every PR. Integration tests are skipped in CI (no live daemon). Add a CI badge to README.md.

## Tasks

### Task 1: Create .github/workflows/test.yml
- Trigger: push to main, pull_request
- Runner: ubuntu-latest
- Checkout with submodules: recursive (vendor/gastown needed for bash function tests)
- Setup Node.js 20
- Run: npm test
- GSD_TOWN_INTEGRATION not set (defaults to 0 — integration tests skipped)

### Task 2: Add CI badge to README.md
- Badge pointing to test.yml workflow on main branch
- Place near top of README, after the title/description line

## Success Criteria

- .github/workflows/test.yml exists and is valid YAML
- Workflow triggers on push/PR, checks out submodules, uses Node 20, runs npm test
- README.md has CI badge with correct workflow URL
- No integration test env var set in CI
